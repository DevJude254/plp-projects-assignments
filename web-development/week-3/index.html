About CSS

CSS (Cascading Style Sheets) is used to style and format HTML elements. With CSS, you can change colors, spacing, fonts, and control the layout of your web pages.

The Box Model

Every HTML element is a box consisting of:

  • Content: The text or image inside the element
  • Padding: Space between the content and the border
  • Border: Wraps around the padding
  • Margin: Space outside the border
